# C Equal language base

## Tokens

### Keywords

#### Conditions
- `if`
- `else`

#### Loops
 - `while`

#### Functions
- `<type> identifier ( <type> identifier, ... )` 

#### Types

*In C=, there are no types, only identifiers of the size of the type*
 - `t8`
 - `t16`
 - `t32`
 - `t64`

#### Operators
- `+`
- `-`
- `*` ?
- `/` ?
- `=`
- `==`
- `!'

## Syntax

### Comments
- `//` Single line comment
- `/*` Multi line comment `*/`

### Variables
- `<type> identifier = value;`

### Functions
- `<type> identifier ( <type> identifier, ... ) { ... }`

### Conditions
- `if ( <expression> ) { ... }`

### Loops
- `while ( <expression> ) { ... }`
